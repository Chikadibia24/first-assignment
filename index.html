My Reflective Documentation:

In the course of building this web page, I learned some valuable skills and lessons.

I was able to make the page responsive using the CSS display properties, such as display: grid, display: flex, and also, display: block, depending on the elements and how I want them to appear and adapt on the viewport.

I used the display: grid and display: flex properties to make my page responsive and adapt to different screen sizes.

I used the display: block and align-items: center to ensure the elements with this CSS property are centered perfectly within their parent elements or their containers.

I then created two media queries for different screen sizes to make my web page more responsive and also make the responsiveness smoother.

Also, I used many HTML tags such as the paragraph tags, the div tags, the section tags, the main tags, the anchor tags, etc. to create and organize my web page.
